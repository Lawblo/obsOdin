# HTML

Basic setup:
<!--
<!DOCTYPE html>
<html lang="en">

<head>
  <title>TITLE</title>
  <meta charset="UTF8">	
  <link rel="stylesheet" href="styles.css"/>
</head>

<body>
	MAIN HTML CODE 
</body>
</html>
-->

[[CSS]] is used to change the way your HTML looks

## HTML terminology
-   `a` is the **tag**
-   `href` is the **attribute**
-   `index.html` is the **attribute value**
-   `Home` is the **text**.

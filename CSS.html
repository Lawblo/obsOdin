Basic Syntax

General Syntax

selector {property: value}

Specific Selector Syntax

Universal Selector:

* {property: value}

Type Selector:

ex. div:

div {property: value}

Class Selector:

In the html: element class="class-name"

In the CSS: .class-name {property: value;}

  • Not required to be unique, so can be used on many elements.
  • Multiple classes can be added to a single element as a space-separated list. ex: class="class1 class2"
  • Never use spaces in class names
